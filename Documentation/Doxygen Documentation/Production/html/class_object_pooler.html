GameObject ObjectPooler.GetObject ( )

Returns the first inactive gameObject in the object pool. If the pool is allowed to grow, and this method fails to return an inactive gameObject, then a new object will be created, added to the pool, and returned.

{GameObject} First inactive object in pool, or newly added gameObject if the pool is allowed to grow.
